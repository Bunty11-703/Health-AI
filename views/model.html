Model Architecture & Evaluation

This page provides an overview of the machine learning model used in HealthAI, including training methodology, evaluation strategy, and known limitations.

HealthAI uses a Random Forest Classifier, an ensemble learning algorithm that combines multiple decision trees to improve prediction accuracy and robustness.

Random Forest was chosen for its ability to handle non-linear feature interactions and provide built-in feature importance for explainability.
